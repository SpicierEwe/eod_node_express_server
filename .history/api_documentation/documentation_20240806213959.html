Ior Management API Documentation

Welcome to the API documentation for the Ior Management System. Below you'll find details on how to use each endpoint.

GET /

Returns this documentation.

curl -X GET http://localhost:3000/

POST /eod

Creates a new Ior entry.

Request Body:

{
    "profile": "Profile Info",
    "name": "Name",
    "months": 6,
    "date": "2024-07-01",
    "issued_by": "Issuer Name",
    "pronoun1": "He",
    "pronoun2": "Him",
    "content": "Ior Content",
    "status": "Active",
    "time_stamp": "2024-07-01 08:00:00"
}
curl -X POST http://localhost:3000/eod -H "Content-Type: application/json" -d '{ "profile": "Profile Info", "name": "Name", "months": 6, "date": "2024-07-01", "issued_by": "Issuer Name", "pronoun1": "He", "pronoun2": "Him", "content": "Ior Content", "status": "Active", "time_stamp": "2024-07-01 08:00:00" }'

GET /eod

Fetches all Ior entries.

curl -X GET http://localhost:3000/eod

GET /eod/:id

Fetches a specific Ior entry by ID.

curl -X GET http://localhost:3000/eod/3

PUT /eod/:id

Updates an existing Ior entry by ID.

Request Body:

{
    "profile": "Updated Profile Info",
    "name": "Updated Name",
    "months": 12,
    "date": "2024-07-02",
    "issued_by": "Updated Issuer Name",
    "pronoun1": "They",
    "pronoun2": "Them",
    "content": "Updated Ior Content",
    "status": "Inactive",
    "time_stamp": "2024-07-02 08:00:00"
}
curl -X PUT http://localhost:3000/eod/3 -H "Content-Type: application/json" -d '{ "profile": "Updated Profile Info", "name": "Updated Name", "months": 12, "date": "2024-07-02", "issued_by": "Updated Issuer Name", "pronoun1": "They", "pronoun2": "Them", "content": "Updated Ior Content", "status": "Inactive", "time_stamp": "2024-07-02 08:00:00" }'

DELETE /eod/:id

Deletes an Ior entry by ID.

curl -X DELETE http://localhost:3000/eod/3

Database Schema

The table structure for the Ior table:

CREATE TABLE Ior (
  id SERIAL PRIMARY KEY,
  profile TEXT,
  name VARCHAR(255),
  months INT,
  date DATE NOT NULL,
  issued_by VARCHAR(255),
  pronoun1 VARCHAR(50),
  pronoun2 VARCHAR(50),
  content TEXT,
  status VARCHAR(50),
  time_stamp TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
