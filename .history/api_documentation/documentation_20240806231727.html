EOD Management API Documentation

Welcome to the API documentation for the EOD Management System. Below you'll find details on how to use each endpoint.

GET /

Returns this documentation.

curl -X GET http://localhost:3000/

POST /eod

Creates a new EOD entry.

Request Body:

{
    "title": "Title",
    "description": "Description of the entry",
    "department": "Department Name",
    "date": "2024-07-01",
    "to": "Recipient Name",
    "from": "Sender Name",
    "createdAt": "2024-07-01 08:00:00",
    "updatedAt": "2024-07-01 08:00:00",
    "employee_id": "12345",
    "manager_id": "67890"
}
curl -X POST http://localhost:3000/eod -H "Content-Type: application/json" -d '{ "title": "Title", "description": "Description of the entry", "department": "Department Name", "date": "2024-07-01", "to": "Recipient Name", "from": "Sender Name", "createdAt": "2024-07-01 08:00:00", "updatedAt": "2024-07-01 08:00:00", "employee_id": "12345", "manager_id": "67890" }'

GET /eod

Fetches all EOD entries.

curl -X GET http://localhost:3000/eod

GET /eod/:id

Fetches a specific EOD entry by ID.

curl -X GET http://localhost:3000/eod/3

PUT /eod/:id

Updates an existing EOD entry by ID.

Request Body:

{
    "title": "Updated Title",
    "description": "Updated Description",
    "department": "Updated Department",
    "date": "2024-07-02",
    "to": "Updated Recipient",
    "from": "Updated Sender",
    "createdAt": "2024-07-02 08:00:00",
    "updatedAt": "2024-07-02 08:00:00",
    "employee_id": "54321",
    "manager_id": "09876"
}
curl -X PUT http://localhost:3000/eod/3 -H "Content-Type: application/json" -d '{ "title": "Updated Title", "description": "Updated Description", "department": "Updated Department", "date": "2024-07-02", "to": "Updated Recipient", "from": "Updated Sender", "createdAt": "2024-07-02 08:00:00", "updatedAt": "2024-07-02 08:00:00", "employee_id": "54321", "manager_id": "09876" }'

DELETE /eod/:id

Deletes an EOD entry by ID.

curl -X DELETE http://localhost:3000/eod/3

Database Schema

The table structure for the eod table:

CREATE TABLE eod (
  id SERIAL PRIMARY KEY,
  title VARCHAR(255),
  description TEXT,
  department VARCHAR(255),
  date DATE NOT NULL,
  to VARCHAR(255),
  from VARCHAR(255),
  createdAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updatedAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  employee_id VARCHAR(255),
  manager_id VARCHAR(255)
);
